# CLASSIFICATION

Classification in Machine Learning

Definition:
Classification is a supervised machine learning algorithm used to predict a category or class based on input data.

Types of Classification in Machine Learning
1. Binary Classification
Predicts 2 classes.
Examples: Yes/No, Pass/Fail, Spam/Not Spam.
2. Multi-Class Classification
Predicts more than 2 classes.
Examples: Grade (A, B, C), Fruit (Apple, Mango, Orange).
3. Multi-Label Classification
One data item can belong to multiple classes at the same time.
Examples: Movie → Action + Comedy, Image → Dog + Person.
4. Imbalanced Classification
One class has many more samples than the other.
Example: Fraud Detection (99% Genuine, 1% Fraud).
Classification in Machine Learning

Definition:

-Logistic Regression – Used for binary classification.
-Decision Tree – Easy to understand and interpret.
-K-Nearest Neighbors (KNN) – Classifies based on nearest neighbors.
-Naïve Bayes – Uses probability for prediction.
-Support Vector Machine (SVM) – Finds the best boundary between classes.
-Random Forest – Combines multiple decision trees for better accuracy.
